
### Set Shortcuts for Intput-Source

```bash
# After install plistedit-pro (in Brewfile)
$ open ~/Library/Preferences/com.apple.symbolichotkeys.plist
```

and set:
- 60 -> NO
  - Disable "Select the previous input source"
- 61 -> YES, 32, 49, 131072
  - Map "Select next source in Input menu" to `Shift + Space`

### Set Shortcuts for Spotlight

``` bash
# After install plistedit-pro (in Brewfile)
$ open ~/Library/Preferences/com.apple.symbolichotkeys.plist
```

and set:
- 64 -> YES, 32, 49, 262144
  - Map "Show Spotlight search" to `Ctrl + Space`
